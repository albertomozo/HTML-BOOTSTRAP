Bootstrap

Bootstrap fue desarrollado por Mark Otto y Jacob Thornton en Twitter, y lanzado como un producto de código abierto en agosto de 2011 en GitHub.

¿Que es Bootstrap?

  • Bootstrap es un "front-end framework" libre para el desarrollo web más rápido y más fácil
  • Bootstrap incluye plantillas de diseño basadas en HTML y CSS para tipografía, formas, botones, tablas de navegación, modales, carruseles de imágenes y muchas otras, así como complementos opcionales de JavaScript
  • Bootstrap también le da la posibilidad de crear fácilmente diseños responsives

¿Por qué utilizar Bootstrap?

  • Fácil de usar: Cualquier persona con conocimientos básicos acaba de HTML y CSS puede comenzar a usar Bootstrap
  • Características responsive: CSS responsive de Bootstrap ajusta a los teléfonos, tabletas y ordenadores de sobremesa
  • Mobile-primera aproximación: En Bootstrap, estilos móviles primero son parte del marco básico
  • Compatibilidad del navegador: Bootstrap es compatible con todos los navegadores modernos (Chrome, Firefox, Internet Explorer, Safari y Opera)

¿Dónde obtener Bootstrap?

  • Descargar Bootstrap de getbootstrap.com
  • Incluir Bootstrap de un CDN(Content Delivery Network)

¿Cómo funciona?

El sistema de cuadrícula de Bootstrap utiliza una serie de contenedores, filas y columnas para diseñar y alinear el contenido. Es construido con flexbox y es totalmente responsive.

  • Los contenedores proporcionan un medio para centrar el contenido de su sitio. Utilice .container para ancho fijo o .container-fluid para todo el ancho.
  • Las filas son grupos horizontales de columnas que aseguran que sus columnas estén alineadas correctamente. Utilizamos el método de margen negativo en .row para asegurar que todo su contenido esté alineado correctamente por el lado izquierdo.
  • El contenido debe colocarse dentro de columnas, y sólo las columnas pueden ser hijos inmediatos de las filas.
  • Gracias a flexbox, las columnas de cuadrícula sin un ancho de conjunto automáticamente se mostrarán con iguales anchos. Por ejemplo, cuatro instancias de .col-sm serán automáticamente cada 25% de ancho para puntos de interrupción pequeños.
  • Las clases de columnas indican el número de columnas que desea utilizar de las 12 posibles por fila. Por lo tanto, si desea tres columnas de igual ancho, puede utilizar .col-sm-4.
  • Los anchos de columna se establecen en porcentajes, por lo que siempre son fluidos y dimensionados en relación con su elemento padre.
  • Las columnas tienen relleno horizontal para crear los canales entre las columnas individuales, sin embargo, puede quitar el margen de las filas y el relleno de las columnas con .no-canaletas en el.
  • Hay seis niveles de rejilla, uno para cada punto de interrupción sensible: todos los puntos de interrupción (extra pequeño), pequeño, medio, grande y extra grande.
  • Los niveles de cuadrícula se basan en anchos mínimos, lo que significa que se aplican a ese nivel y todos los que están por encima (por ejemplo, .col-sm-4 se aplican a dispositivos pequeños, medianos, grandes y extra grandes).
  • Puede utilizar clases de cuadrícula predefinidas o mezclas Sass para obtener más marcado semántico.
